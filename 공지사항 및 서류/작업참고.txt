스크립트 작성 -> 취합 후 검수 -> 수정 -> 제출 순으로 진행
단어 1개당 지문 3개를 작성하면 되고,
매주 단어 100개씩 작성하면 됨.(지문 총 300개)

단어는 자주 쓰이는 단어가 아니더라도 스크립트로 작성할 수 있는 단어면 그냥 사용.
(자주 쓰이는 단어만 사용하면 단어가 부족함ㅠㅠ 흑흑)
또한 아래의 리스트에 있는 단어는 사용하면 안됨.

== 중복 확인 리스트
전체 외래어 리스트_잉헌사용분(sheet2)
일반 외래어 1,2,3차 리스트

작성 후 아래에 올리기
접속 주소: langtech.synology.me
ID: hufs_nia
PW: 2020nia!

검수는 A/B조 나눠서 하고 장민지는 4차 작업시 A조, 5차 작업시 B조임.
검수시 수정해야하는 셀은 파란색, 제거해야하는 셀은 빨간색으로 칠하고 메모 남김.
(이는 추후에 추가로 설명)

=========================

중분류 => 유래어 ex)카라테 -> 일본어
소분류 => 카테고리 ex)카라테 -> 체육
상황 => 외래어 한글표기(원어표기) ex) 가라테(唐手)

모두 '우리말샘'을 기준으로 작성하고,
중/소분류가 안 나와 있다면 비워둬도 됨.
ex) 지명/인명 같은 경우는 소분류 없음.

외래어 한글표기는 우리말샘 기준이지만 ^,-는 사용하지 않고 띄어쓰기만 사용.
띄어쓰기는 원어표기의 띄어쓰기를 따르되 작성자의 기준에 따라 다르게 써도 됨.
ex) 다크 서클(dark circle) 이지만 다크서클(dark circle)로 써도 됨.

외래어가 축약형이라면 원어표기시 축약형과 풀어서 같이 씀.(우리말샘에 나와 있음)
ex) 디엠(DM/direct message)

외래어가 원어를 생략 또는 축약해서 한글로 표기하더라도 원어는 전체를 표기
ex) 다이아(diamond)

콩글리시의 경우는 발음을 표기
ex) 선팅(sun tinting)

영어->일본식 발음->외래어 의 과정으로 만들어진 외래어는 자주 쓰인다면 포함
원어는 영어를 기재
ex) 공구리(concrete)

한자나 일본어가 쓰이는 경우는 원어를 한자나 일어를 기재
ex) 스모(sumô[相撲/角力]) => 스모(角力)

지문은 작성 후 부산대 맞춤법 검사기로 검사하고 수정 후에도 본인 분량 검사.
